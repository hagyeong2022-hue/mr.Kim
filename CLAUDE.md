# 김비서 — 업무 대시보드

비즈니스 업무 관리를 위한 올인원 웹 기반 대시보드.

## 🎯 주요 기능

- **할 일 관리** - 작업 목록 추가 및 진행 상황 추적
- **일정 관리** - 회의 및 일정 계획
- **매출 현황** - 실시간 매출 데이터 시각화
- **다이어그램** - 업무 흐름 및 프로세스 다이어그램

## 📁 파일 구조

```
├── dashboard.html              # 메인 대시보드 페이지
├── 업무대시보드.html            # 한글 버전 대시보드
├── 시작하기.txt                 # 빠른 시작 가이드
├── 치트시트.txt                 # 주요 기능 정리
├── 1-설치가이드/               # 설치 관련 문서
│   ├── Mac-설치.txt
│   └── Windows-설치.txt
├── 2-체험용-샘플/               # 샘플 데이터
│   ├── 매출보고서.html
│   ├── 마케팅-워크플로우.svg
│   └── 회의록.txt
├── 3-도구만들기/                # 커스텀 도구 가이드
│   └── 프롬프트-예시.txt
├── 4-자동화/                    # 자동화 기능
│   └── 자동화-예시.txt
└── 5-영상만들기/                # 영상 제작 가이드
    └── Remotion-가이드.txt
```

## 🛠 사용 기술

- **HTML5** - 구조 및 마크업
- **CSS3** - 스타일링 및 레이아웃
- **JavaScript** - 대화형 기능 및 상태 관리
